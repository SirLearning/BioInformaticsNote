1.按需求从文件中选择数据输出到下一个文件
```
cat example|awk '{if ($2>5000) print $1"\t" $7 }'>f1
awk 结构:查看该文件（cat） |awk (模式 -v，-f等) ‘{命令内容}’  > 输出到的文件
```
2.    排序
```
sort -n -k 2 f1 > f2
sort 排序命令，-n表示从小到大排，-k表示区域选择，2表示行号，f1是待处理文件
```
3.    统计：
```
wc -l 统计行数  -w 统计字符串数 -c 统计字节数， 要与cat file 连用（管道符|）
```
4.    清屏：
```
clear；不需要任何参数
NR 当前文件中的行数
```
